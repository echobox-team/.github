# Echobox Team

É um grupo acadêmico dedicado ao desenvolvimento do Echobox, um sistema de feedback inspirado no Reclame Aqui, mas projetado com foco em usabilidade, clareza e experiência visual aprimorada.

O projeto foi desenvolvido nas disciplinas de Web Development Framework e Backend: Webservices, aplicando boas práticas de arquitetura, qualidade de código e evolução técnica ao longo do processo.

## Projetos da Organização

### 1. [`EchoBox` — Monorepo (React.js + Node.js)](https://github.com/echobox-team/EchoBox)

Repositório completo do projeto original, desenvolvido para a disciplina de Web Development Framework.

Inclui:

* **Frontend em React.js** focado em experiência do usuário
* **API em Node.js** construída para complementar o sistema
* **Banco de dados PostgreSQL local**
* **Sistema completo**, com:

  * Fluxo do usuário final
  * Versão para empresas
  * Painéis e CRUDs administrativos

Todo o projeto foi pensado com foco em **usabilidade**, acessibilidade e funcionamento real de um sistema de feedback.

---

### 2. `Echobox_java` — API em Spring Boot

Reescrita da API do Echobox para a disciplina de Backend: Webservices, agora utilizando **Java + Spring Boot**.

Destaques:

* API mais robusta e estruturada
* Documentação automática via **Swagger**
* Controllers e Services com **testes unitários implementados**
* Autenticação com **JWT**
* Evolução clara de arquitetura e boas práticas em relação à primeira versão em Node.js

Esse repositório demonstra o amadurecimento do time em padrões, organização e qualidade de código.
